This module implements a simple homepage banner slideshow similar to the one on the Iowa 
State University homepage (http://www.iastate.edu/). It presents a views slideshow in a 
block.

INSTALLATION

Dependencies:
	ctools
	features
	image (core)
	jquery_update
	libraries
	link
	views
	views_slideshow

Install jquery.cycle in /sites/all/libraries/jquery.cycle/jquery.cycle.all.min.js. Find it
at http://jquery.malsup.com/cycle/download.html
